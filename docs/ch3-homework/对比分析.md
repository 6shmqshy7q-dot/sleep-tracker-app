产品想法.md
1. 产品名称
睡眠监测器
2. 产品用途
通过智能设备和手机传感器采集用户睡眠数据（浅睡 / 深睡 / REM 周期等），自动生成睡眠健康报告，并根据数据推送个性化改善建议，帮助用户了解自身睡眠质量、调整作息习惯
3. 4 个重要的现实约束
范围约束：仅面向个人消费者使用，不支持企业 / 医疗机构批量采购和商用部署
交易约束：分为免费版（基础睡眠数据查看）和付费版（深度报告 + 个性化建议），付费版支持日付、月付、年付三种订阅模式，无一次性买断选项
内容约束：睡眠分析数据来源于合作医疗机构的睡眠健康模型，用户数据本地存储，不向第三方泄露，仅用于生成个人报告
运营约束：数据采集依赖蓝牙智能手环 / 手表，或手机内置的加速度传感器，无设备则无法生成完整睡眠报告
4. 涉及到的认知模型（最重要的 4 个实体）
「用户」：核心属性包括姓名、年龄、身高、体重、设备型号等
「睡眠数据记录」：核心属性包括记录 ID、关联用户、睡眠时长、浅睡时长、深睡时长、REM 周期时长、入睡时间、醒来时间、睡眠评分
「订阅订单」：核心属性包括订单 ID、关联用户、订阅类型（免费 / 日付 / 月付 / 年付）、支付金额、生效时间、到期时间、是否自动续费
「睡眠改善方案」：核心属性包括方案 ID、关联用户、关联睡眠记录、建议类型（作息调整 / 环境优化 / 饮食建议）、建议内容、生成时间
