# 使用的AI工具
## 一、基础信息
- 工具名称：Zed built-in AI assistant（Zed内置AI助手）
- 核心模型：Claude 3.5 Sonnet
- 核心能力：支持Markdown格式生成、认知敏捷法文档创作，可调用42cog Skills扩展功能

## 二、文档生成流程
1.  梳理睡眠监测器核心需求，完成《产品想法.md》，明确产品约束与核心实体；
2.  未调用42cog Skills，生成方式A的3份基础文档（real.md/cog.md/spec.md）；
3.  调用42cog Skills，生成方式B的3份专业文档（补充约束优先级、实体关联关系等）；
4.  对比两类文档差异，完成《对比分析.md》，梳理42cog Skills的价值。

## 三、格式说明
- 所有文档均为Markdown格式，符合作业提交规范；
- 核心文件统一存放于目录，便于审核。
