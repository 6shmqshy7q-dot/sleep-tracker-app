# 睡眠监测器网站开发报告

## 项目概述

本项目基于 week3 认知敏捷法作业，成功开发了一个专业的睡眠监测器产品展示网站。网站采用 Next.js + TypeScript + Tailwind CSS 技术栈，实现了完整的产品功能展示和商业化方案。

## 项目信息

- **项目名称**: 智能睡眠监测器产品展示网站
- **开发时间**: 2025-12-15
- **技术栈**: Next.js 16.0.10 + TypeScript + Tailwind CSS
- **配色方案**: 个性+清新风格（薄荷绿+紫色+天空蓝）
- **运行地址**: http://localhost:3000

## 核心设计理念

基于 week3 认知敏捷法的 4 个核心实体设计：

### 1. 用户 (User)
- 睡眠监测器的核心用户群体
- 关注睡眠健康的个人消费者
- 年龄范围：18-65岁

### 2. 睡眠数据记录 (Sleep Data Record)
- 浅睡/深睡/REM周期数据
- 睡眠时长、入睡时间、醒来时间
- 睡眠质量评分（0-100分）

### 3. 订阅订单 (Subscription Order)
- 免费版：基础睡眠数据查看
- 付费版：深度分析报告 + 个性化建议
- 年付版：所有功能 + 优先支持

### 4. 睡眠改善方案 (Sleep Improvement Plan)
- 作息调整建议
- 环境优化建议
- 饮食运动建议

## 网站结构

### 1. 导航系统
- 首页：产品介绍和核心理念
- 核心功能：四大功能模块展示
- 睡眠报告：真实数据报告示例
- 改善建议：个性化建议系统
- 订阅方案：三种付费模式

### 2. 核心功能模块

#### 数据采集系统
- 支持蓝牙智能手环/手表和手机传感器
- 自动采集睡眠数据，精度高达95%以上
- 技术标签：蓝牙4.0+、传感器融合、实时采集

#### 睡眠分析引擎
- 基于合作医疗机构睡眠健康模型
- 深度分析浅睡/深睡/REM周期，生成科学评分
- 技术标签：AI算法、医疗模型、多维分析

#### 个性化建议系统
- 根据睡眠数据生成个性化改善方案
- 涵盖作息调整、环境优化、饮食运动等
- 技术标签：机器学习、知识图谱、智能推荐

#### 订阅服务管理
- 灵活的付费模式：免费版查看基础数据，付费版享受深度报告
- 技术标签：支付集成、权限管理、自动续费

### 3. 睡眠报告展示
展示真实睡眠数据，包括：
- 日期和睡眠质量评级（优秀/良好/中等）
- 睡眠时间、总时长、睡眠评分
- 深睡时长等关键指标

### 4. 个性化改善建议
基于AI分析的个性化建议：
- 作息调整：建议提前30分钟入睡
- 环境优化：卧室温度和湿度控制
- 饮食建议：晚餐时间和内容调整

### 5. 订阅方案
三种灵活的订阅模式：

#### 免费版 (¥0/永久)
- 基础睡眠数据查看
- 最近30天数据
- 简单睡眠评分

#### 付费版 (¥29.9/月) - 推荐
- 深度睡眠分析报告
- 个性化改善建议
- 历史数据趋势
- 睡眠模式对比

#### 年付版 (¥299/年)
- 所有付费版功能
- 优先客服支持
- 年度睡眠报告
- 健康数据导出

### 6. 产品特色
- 数据采集精度：95%+
- 全天候监测：24/7
- AI智能分析引擎
- 用户信赖选择：100K+

## 技术实现

### 前端技术栈
- **Next.js 16.0.10**: React框架，支持服务端渲染
- **TypeScript**: 类型安全的JavaScript超集
- **Tailwind CSS**: 实用程序优先的CSS框架
- **next-themes**: 主题切换功能

### 设计特色
- **配色方案**: 薄荷绿主色调 + 紫色辅助 + 天空蓝点缀
- **响应式设计**: 完美适配桌面和移动设备
- **明暗主题**: 支持系统偏好或手动切换
- **交互效果**: 悬停动画、渐变背景、装饰线条

## 部署说明

### 本地开发
1. 进入项目目录：`cd personal-website`
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 访问：http://localhost:3000

### 生产部署
1. 构建项目：`npm run build`
2. 启动生产服务器：`npm start`
